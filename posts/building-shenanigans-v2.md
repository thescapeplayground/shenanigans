## Stack

- **Next.js** — app router, server components where it makes sense
- **Tailwind CSS v4** — utility-first, fast iteration
- **Framer Motion** — page transitions and micro-interactions
- **shadcn/ui** — component primitives, customized

## Design Goals

1. **Minimal** — no fluff, just the content
2. **Dark mode first** — but light mode exists too
3. **Fast** — lightweight, no unnecessary JS
4. **Personal** — reflects the vibe, not a template

## What's New

The gallery is more organized, the stack section shows tools I actually use, and now — a blog.
